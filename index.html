How to Play

Welcome to the Merlin Game! Here's how to play:

  • Start the game: Press any button to begin.
  • Follow the pattern: The game will light up buttons in a sequence, and you must repeat it.
  • Level up: Each time you correctly follow the sequence, it gets longer and more complex.
  • Winning: Keep up with the sequences as long as you can to score high!

Good luck, and have fun!
